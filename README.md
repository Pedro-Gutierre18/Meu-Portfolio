# 🎯Meu Portfólio Pessoal
Bem-vindo ao meu portfólio! <br>
Aqui falo um pouco sobre de mim, exibo minhas habilidades e alguns de meus projetos.

## 🔎Sobre o Projeto:
Foi muito divertido criar esse portfólio, criando ele, melhorei minhas habilidades e organização. 
- Criei o escopo do site usando o `Figma`, lá consegui definir coisas como, paleta de cores, fontes e sessões; 
- Utilizei `display:grid` para criar os cards das sessões habilidades e projetos; 
- Nesse site utilizei recursos de acessbilidade para definicientes visuais; 
- Usei a API `ScrollReveal` para adicionar animações conforme o usuario desce a página; 
- Também usei a API `EmailJS` para dar funcionalidade ao formulário de contato.
